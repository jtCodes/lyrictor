# Text Customization Ideas

Current text-item customization already covers:

- Font family
- Font size
- Font weight
- Font color
- Shadow blur
- Shadow color
- Item opacity
- X and Y offset
- Text effects: Ash Fade, Blur, Glitch

Gaps that would make text feel less generic without relying on font changes:

## High-Value Additions

1. Stroke and stroke width
   Add outline controls so text can have a thin dark edge, soft light rim, or colored border.

2. Letter spacing
   Tracking control would help short lyric lines feel more editorial and less default.

3. Line height
   Multi-line lyrics need adjustable vertical rhythm to avoid looking stock.

4. Text alignment inside the box
   Left, center, and right alignment would open up more layout styles.

5. Background backing
   A per-lyric backing style with opacity, blur, padding, and corner radius would help text sit on busy backgrounds.

6. Gradient fill
   A simple two-stop or vertical gradient fill would make text feel more designed than a flat color.

7. Shadow X/Y offset
   Separate offset controls would allow real depth and directional shadowing, not just glow.

8. Text transform
   Uppercase, lowercase, and other casing treatments would add styling flexibility.

9. Text box width and padding
   More control over the block itself would help create less generic compositions.

10. Lightweight entrance emphasis controls
   Small scale-in, blur-in, upward drift, or fade curve controls could add character without requiring a full effect preset.

## Recommended First Batch

If only a few should be built first, the strongest order is:

1. Stroke
2. Letter spacing
3. Line height
4. Background backing
5. Gradient fill
6. Shadow offset

## Why These First

- Stroke immediately improves legibility and style range.
- Letter spacing and line height make text layouts feel intentional.
- Background backing helps text survive on complex imagery.
- Gradient fill and shadow offset add polish without needing heavy animation.
